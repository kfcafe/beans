### parse_agent_event (from ~/deli/src/agent.rs lines 130-230)
Parses a serde_json::Value from pi's JSON output into AgentEvent:
- assistantMessageEvent.type == "thinking_delta" → Thinking
- assistantMessageEvent.type == "text_delta" → Text
- assistantMessageEvent.type == "toolcall_start" → ToolStart (read name from message.content)
- assistantMessageEvent.type == "toolcall_end" → ToolEnd (with arguments)
- type == "turn_end" → TokenUpdate (from message.usage) or ToolResult
- result field present → Finished
