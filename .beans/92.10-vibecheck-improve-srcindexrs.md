## Task
Improve `src/index.rs` based on vibecheck findings.

## Findings (top 20)
- [low] performance — `src/index.rs:82-92`
  - `is_bean_filename` is called for every file in `.beans/` during index builds. It uses string operations and Path parsing. The check `filename.contains('-')` for `.md` files means a file like `notes-for-meeting.md` in `.beans/` would be treated as a bean file, leading to parse errors that are silently swallowed during index builds.
  - Suggested fix: Tighten the pattern: require the filename to start with a digit or known ID format (e.g., `^[0-9]`). This would prevent non-bean markdown files from being mistakenly parsed.

## Notes
- Keep behavior the same unless the finding explicitly calls for a behavior change.
- Prefer small, focused changes.
